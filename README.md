# Sentiment analysis perform on a Financial dataset.
This project perfoms a Sentiment Analysis of a financial dataset for the Unit 5 of Intelligent Systems assignment. The goal of the project is to implement a project based on Natural Language techniques. 

The dataset obtain from Kaggle [Dataset](https://www.kaggle.com/datasets/ankurzing/sentiment-analysis-for-financial-news) contains information about financial new's headlines which are processed to obtain if they are negative, neutral or positive for develep a personal investment. 

Inside the src folder there is a R project and the main.r file must be executed to see the results of the project.
